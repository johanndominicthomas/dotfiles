systemctl
------------
Execute the below commands:

# Display Manager
sudo systemctl enable sddm.service --now


# Audio stack
systemctl --user enable pipewire.service --now
systemctl --user enable wireplumber.service --now
systemctl --user enable pipewire-pulse.service --now

# Bluetooth
sudo systemctl enable bluetooth.service --now

# Power profiles
sudo systemctl enable power-profiles-daemon.service --now

# ASUS laptop controls
sudo systemctl enable asusd.service --now
